囚徒困境
PRISONER'S DILEMMA
为什么理性的人
会做出对双方都不利的选择？
/ 设定
两个嫌疑犯被分开审讯
🧑
囚犯 A
🔒
🧑
囚犯 B
他们无法沟通
每人只有两个选择
🤝 合作（沉默）
🗡️ 背叛（揭发）
/ 收益矩阵
不同选择的结果
B 合作 B 背叛
A 合作 -1 / -1
各判1年		 -10 / 0
A判10年，B释放
A 背叛 0 / -10
A释放，B判10年		 -5 / -5
各判5年
双方合作 → 各判1年（最优）
双方背叛 → 各判5年（最差）
/ 困境
理性的选择……
A 的想法：
「如果 B 合作 → 我背叛得 0年，合作得 1年 → 背叛更好」
「如果 B 背叛 → 我背叛得 5年，合作得 10年 → 还是背叛更好」
B 的想法：
完全一样的推理 → 背叛也更好
⚡ 纳什均衡：双方都背叛 → 各判5年
明明双方合作只需各判1年
但「理性」却把他们推向了最差结局
/ 现实中的囚徒困境
🏭
价格战
两家公司都降价 → 利润全没了，但谁都不敢先涨价
🌍
气候变化
减排对全球有利，但每个国家都想「搭便车」让别人先减
💊
抗生素滥用
个人用抗生素有益，但所有人滥用 → 超级细菌
🚗
堵车 vs 公交
人人开车 → 全堵；人人坐公交 → 全通。但「我一个人开车没关系」
/ 破局之道
重复博弈改变一切
如果游戏只玩一次 → 背叛是最优
如果反复博弈 → 合作策略胜出
😈
永远背叛
每次都背叛 → 短期赢，长期被所有人报复
😇
永远合作
每次都合作 → 容易被人利用
🏆
以牙还牙 (Tit for Tat)
第一回合合作，之后模仿对手上一轮的选择
🎲
随机策略
随机合作或背叛 → 不可预测但也不稳定
/ 结论
善良 · 可激怒 · 宽容
「以牙还牙」策略为什么赢？
善良 — 从不主动背叛
可激怒 — 对背叛立即报复
宽容 — 对方改过就原谅
「在一个可能再次相遇的世界里
合作是最聪明的自私」
—— 罗伯特·阿克塞尔罗德 《合作的进化》
